# 🌆 UrbanAware — Smart Urban Sustainability Platform

UrbanAware is a smart web platform designed to help users understand, monitor, and improve their city’s environmental conditions through real-time data, learning, and community participation.

---

## 🚀 Live Demo

🔗 https://urban-aware.vercel.app/

---

## ✨ Features

### 📊 Live Dashboard
- Real-time city insights:
  - AQI (Air Quality Index)
  - Traffic Levels
  - Temperature
- Interactive map:
  - Click any location to view data
- Dynamic graphs for better visualization

---

### 🧠 Learn & Explore
- Sustainability guidelines (Do’s & Don’ts)
- 📝 Quiz system:
  - Dynamic questions
  - Instant scoring
- 🎯 Scenario-based learning:
  - Real-life problem solving
  - Feedback and performance tracking

---

### 🚗 Drives
- Join existing sustainability drives
- Host your own drives
- Instant participation system

---

### 📝 Report Issues
- Report problems like:
  - Waste
  - Traffic
  - Pollution
- Upload images
- Option for anonymous reporting

---

### 👤 Profile & Gamification
- Track:
  - 🌟 Score
  - 🏆 Badges
  - 🚗 Drives participation
  - 📝 Reports submitted
- Encourages active engagement

---

### 💡 Suggestion System
- Users can suggest improvements
- Helps evolve the platform continuously

---

## 🛠️ Tech Stack

- Frontend: HTML, CSS, JavaScript  
- Backend: Node.js, Express.js  
- Database: MongoDB  
- APIs: WAQI API, OpenWeather API  

---

## 🌍 Purpose

UrbanAware aims to bridge the gap between awareness and action by providing users with tools to learn, participate, and contribute towards building sustainable cities.
